# أداة مقارنة الأسهم | Stock Comparison Tool

أداة تفاعلية لمقارنة أسهم الشركات باستخدام بيانات من Yahoo Finance.

## المميزات

- مقارنة سهمين جنب بعض
- بيانات مالية كاملة (P/E، هامش الربح، ROE، إلخ)
- شارت سعر 10 سنوات
- شارت الأرباح والإيرادات (ربع سنوي / سنوي)
- مقارنة بصرية بـ Radar Chart
- جدول مقارنة تفصيلي
- شرح لكل مؤشر ( tooltips )
- تصميم عصري ومتجاوب

## التقنيات

- React + Vite
- Tailwind CSS
- Recharts
- Express ( backend )
- yahoo-finance2

## التشغيل

### 1. ثبّت التبعيات
```bash
npm install
```

### 2. شغّل السيرفر (نافذة أولى)
```bash
npm run server
```

### 3. شغّل الواجهة (نافذة ثانية)
```bash
npm run dev
```

### 4. افتح المتصفح
```
http://localhost:5173
```

## ملاحظات

- البيانات من Yahoo Finance وليست بيانات لحظية من بورصة رسمية
- لأغراض تعليمية وبحثية فقط وليست نصيحة استثمارية
